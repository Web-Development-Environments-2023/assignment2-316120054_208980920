About The Game

Hello everyone!


  • This is a game about a starship that is trying to kill all the bad enemies.
  • The starship is controlled by the arrow keys.
  • The goal is to get to the end of the game by killing all the enemies.
  • The enemies are trying to kill the starship.
  • The starship can shoot bullets to kill the enemies.
  • You can choose a button to shoot the enemies and choose also the duration of the game at the configuration window.
  • The StarShip starts the game with 3 lives.
  • We hope you will enjoy the game.

    • Good luck!
